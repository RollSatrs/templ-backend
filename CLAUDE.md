# Правила проекта

Это шаблон, на основе которого запускаются новые проекты. Правила ниже действуют
по умолчанию для любой задачи в этом репозитории, если пользователь явно не попросил иначе.

## Изменения схемы БД (Drizzle)

- Никогда не применяй `drizzle-kit push` для реальных изменений схемы — он меняет
  базу напрямую, без файла миграции и истории изменений.
- Правильный порядок для любого изменения `src/db/schema.ts`:
  1. `pnpm db:generate` — сгенерировать файл миграции в `drizzle/`
  2. `pnpm db:migrate` — применить миграцию к базе
- `pnpm db:push` допустим только для быстрого локального эксперимента с схемой,
  который потом всё равно нужно оформить через `db:generate` перед коммитом.
- Не редактируй уже сгенерированные файлы миграций вручную и не удаляй их из `drizzle/meta`.
